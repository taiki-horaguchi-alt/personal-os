# Decision Framework

## 汎用判断フロー
1. **Fact Check:** まず事実を集める。「誰かの意見」と「事実」を峻別する。
2. **Options:** 選択肢を最低3つ出す。（松・竹・梅、あるいは現状維持・積極策・撤退）
3. **Criteria:** 「優先順位ルール.md」と照らし合わせる。
4. **Draft:** 結論を仮置きし、一晩寝かす。（感情的なバイアスを除くため）

## 判断に迷った時の問い
* **「もし失敗したら、最悪どうなるか？」** (Worst Case Analysis) - それが許容範囲ならGO。
* **「5年後の自分が今の自分になんて言うか？」** (Long-term Perspective)
* **「尊敬する〇〇さんならどうするか？」** (Role Model)
* **「これをやらない理由は『恐怖』か『面倒』か？」** - 恐怖ならやるべき、面倒なら自動化すべき。
